Er is helaas geen berichtgeving bij Coding the Curbs alleen dat houdt niet dat ze het niet nodig hebben. Ik zelf denk dat als ze een berichtgeving hebben voor het versturen van een bericht via de contact pagina het voor eindgebruikers veel fijner zal zijn om gebruik te maken van de opties

Toast(Pop Up Berichtgeving)

Klik op het knop voor een voorbeeld dat 5 seconden zal duren.

De bovenste code's worden gebruikt voor de -webkit-animation.


JS


function launch_toast() {
	var x = document.getElementById("toast")
	x.className = "show";
	setTimeout(function(){ x.className = x.className.replace("show", ""); }, 5000);
  }

Dit wordt weer gebruikt voor het animatie overgang van de berichtgeving.
