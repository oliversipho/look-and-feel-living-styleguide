Het is handig om kleuren in variabelen in de root te zetten. Dan staan alle kleuren bij elkaar en zijn ze makelijk aan te passen.

Hieronder staat een voorbeeld van hoe je de kleuren in de root kan zetten. 
					
	:root {
		--primary-color: #524dd0;
		--accent-color: #00ffec;
		--gray-bg: #f7f7f7;
	}


	main {
		background: var(--primary-color);
	}
