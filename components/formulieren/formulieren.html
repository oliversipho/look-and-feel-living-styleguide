Omschrijving

De input velden liggen op een lichtgrijze achtergrond. De inputvelden zelf zijn wit met een iets donkere kleur grijs als tekst voor in de input velden.

Kleuren

#f7f7f7
#c9c7c8
#ffffff
#fba5a4

Gedrag

Wanneer je een input veld selecteert wordt de rand er omheen zwart zodat je goed ziet dat je kan gaan typen in het input veld.

Wanneer er een veld verplicht is om ingevuld te worden zal de kleur rood worden wanneer er niets wordt ingevuld
