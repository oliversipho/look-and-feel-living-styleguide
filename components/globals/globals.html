Desktop header

Services Updates Over ons Contact

Gebruik:

Deze header wordt gebruikt als hoofdnavigatie van de desktop website van CTC. De koppen kunnen vervangen worden voor verschillende delen van de site. Er moet gebruik gemaakt worden van media breakpoints om de header responsive in te zetten. De knoppen verwijzen naar de contact pagina en een functie om de taal van de weergeven content te veranderen.

HTML code voorbeeld:

				
					<header>
						<img src="./assets/CTC_Primary_Logo_White.png" alt="brand logo">
						<nav>
							<span>Services</span>
							<span>Updates</span>
							<span>Over ons</span>
							<span>Contact</span>
							<button>Kom in contact ></button>
							<button>NL <span class="move">∨</span></button>
						</nav>
					</header>

CSS code voorbeeld: 

				
					header{
						padding: 1.3em;
						background-color: #524dd0;
						padding-top: 2em;
						font-family: 'Rajdhani', sans-serif;
						color: white;
						display: flex;
						flex-flow: row nowrap;
						justify-content: flex-end;
						font-size: 18px;
					}
					
					header > img{
						max-width: 9em;
						margin-right: auto;
						margin-left: 3.5em;
					}
					
					
					header > div > span{
						margin-right: 4em;
					}
					
					header > div > span:hover{
						color: black;
						transition: .3s ease;
					}
					
					header > div > button:nth-of-type(1){
						font-family: 'Rajdhani', sans-serif;
						font-size: 18px;
						color: black;
						background-color: #00ffec;
						border: none;
						padding: .75em 2.5em .75em 2.5em;
						border-radius: .5em;
					}
					
					header > div > button:nth-of-type(1):hover{
						background-color: white;
						transition: .3s ease;
					}
					
					header > div > button:nth-of-type(2){
						padding: 1em;
						color: white;
						background-color: #524dd0;
						border: solid white 1px;
						border-radius: .25em;
						margin-left: .2em;
					}
					
					header > div > button:nth-of-type(2):hover{
						background-color: #242e73;
					}
					
					.move{
						margin-left: 1em;
					}

Mobiele Header

Gebruik:

Deze header wordt gebruikt als mobiele header en heeft geen informatie buiten het logo om. Dit is de header die automatisch aangenomen wordt onder een bepaalde schermgrootte.

HTML code voorbeeld:

				
					<header>
						<img src="./assets/CTC_Primary_Logo_White.png" alt="brand logo">
					</header>

CSS code voorbeeld: 

				
					header{
						padding: 1.3em;
						background-color: #524dd0;
						padding-top: 2em;
						display: flex;
						font-size: 18px;
						max-width: 40em;
					}
					
					header > img{
						max-width: 9em;
						margin-right: auto;
						margin-left: 3em;
					}

Desktop Footer

Services Over ons Contact

Gebruik:

De footer wordt op desktop pagina's ingezet als globaal element dat belangrijke informatie bevat over het bedrijf en hoe je hier contact mee kan opnemen. Veel van de informatie staat ook in de header met uitzondering van de verschillende partner foto's die bepaalde connecties weergeven.

HTML code voorbeeld: 

				
					<footer>
						<img src="./assets/CTC_Primary_Logo_White.png" alt="brand logo">
						<div>
							<span><a href="">Services</a></span>
							<span><a href="">Over ons</a></span>
							<span><a href="">Contact</a></span>
						</div>
						<div>
							<img src="./assets/Logo White RGB.webp" alt="brand logo">
							<img src="./assets/Logo FMN white.webp" alt="brand logo">
						</div>
						<button>Kom in contact ></button>
					</footer>

CSS code voorbeeld: 

				
					footer{
						padding-bottom: 8em;
						background-color: #09102E;
						padding-top: 2em;
						display: flex;
						flex-flow: row nowrap;
						color: white;
					}
					
					footer > img{
						max-width: 7em;
						height: 2.5em;
						margin-right: auto;
						margin-left: 3em;
					}
					
					footer > div{
						display: flex;
						flex-flow: column nowrap;
						margin-right: 5em;
						font-family: 'Rajdhani', sans-serif;
					
					}
					
					footer > div > span{
						margin-bottom: 1em;
					}
					
					footer > div > span > a:visited{
						color: white;
					}
					
					footer > div > img{
						margin-bottom: 1em;
					
					}
					
					footer > button{
						font-family: 'Rajdhani', sans-serif;
						font-size: 18px;

						height: 2.7em;
						margin-right: 3em;
					}
					
					footer > button:hover{
						background-color: white;
						transition: .3s ease;
					}
